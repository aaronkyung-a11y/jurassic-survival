## 🎮 플레이하기 · Play

- **GitHub Pages**: 이 레포에서 `Settings → Pages → Source: main → / (root)` 로 설정하면 자동으로 배포됩니다. (`https://<username>.github.io/<repo-name>/`)

## ✨ 주요 기능 · Features

### 공룡 생태계
- **육식공룡** — 시야에 플레이어가 들어오는 즉시 추격. 콤프소그나투스(떼), 벨로시랩터, 딜로포사우루스(독액), 카르노타우루스(돌진).
- **초식공룡** — 평소엔 배회하지만 공격받으면 격분. 파라사우롤로푸스, 트리케라톱스(돌진), 스테고사우루스(가시꼬리 방사 공격), 안킬로사우루스(지면 강타 충격파).
- **보스전 3회** — 3구역 T-Rex, 6구역 Spinosaurus, 10구역 Indominus Rex (투명 망토 페이즈).

### 무기 시스템
| 무기 | 특징 |
|---|---|
| `1` 택티컬 라이플 | 밸런스형 기본 무기 |
| `2` 산탄총 | 7펠릿 부채꼴 확산, 근접 고데미지 |
| `3` 기관단총 | 매우 빠른 연사, 탄막 뿌리기 |
| `4` 쌍권총 | 양손 교차 발사, 빠른 재장전 |
| `5` 저격소총 *(DNA 240)* | 관통 3 · 초고데미지 · 장거리 카운터 |
| `6` 화염방사기 *(DNA 340)* | 다단히트 · 광역 · 대량 탄약 |

### 로그라이크 루프
- 구역 클리어 후 3장 랜덤 업그레이드 선택 (HP, 데미지 배율, 연사속도, 탄창, 관통, 이동속도, 탄속, 재장전)
- **DNA 무기고**에서 보스 드랍으로 쌓은 DNA로 신무기 구매
- 보스 처치 시 방사형 DNA 폭발 (T-Rex 140 / Spino 220 / Indominus 400)

### UI · 연출
- 섹터 전환 시 바이옴 이름과 플레이버 텍스트가 담긴 영화적 인트로
- 전환 후 1.5초 (보스는 2.5초) 공룡 정지 준비 시간
- EXIT 개방 배너, 공룡 머리 위 한글 이름표, 모바일 햅틱 피드백
- CRT 스캔라인 + 쥬라기 공원 앰버/블러드/에메랄드 팔레트

## 🎯 조작 · Controls

### 데스크톱
- `WASD` 이동 · `마우스` 조준 · `좌클릭` 발사
- `Space` 구르기 (무적) · `R` 재장전
- `1`~`6` 무기 전환 · `Q`/`E` 순환

### 모바일
- 왼쪽 가상스틱 이동, 오른쪽 가상스틱 조준+발사
- 화면 두 번 탭 구르기
- 상단 무기 버튼 탭으로 전환

## 🛠️ 기술 스택 · Tech

- **순수 HTML/CSS/JavaScript** — 빌드 도구 없음, 외부 런타임 의존성 없음
- **HTML5 Canvas 2D** — 절차적으로 드로잉한 공룡 실루엣, 명암 그라데이션, 월드공간 그림자
- **Google Fonts** — VT323 / Rubik Mono One / Major Mono Display (유일한 네트워크 의존성)
- 싱글 파일 ~110KB, 약 3천 줄

## 🏗️ 개발 노트

Claude Opus 4.7과의 반복 세션으로 제작. 주요 이터레이션:

1. 베이스 게임 (탑다운 로그라이크 루프, 공룡 6종, 보스 3종)
2. 섹터 전환 연출, 공룡 이름표, 트리케라톱스 밸런스, 스테고 특수공격
3. 안킬로 지면 강타, 전환 후 준비 시간, 무기 시스템 (4종)
4. 공룡 넉백, EXIT 배너, 보스 DNA 대폭 증가, 상점 & 신무기 2종 (저격/화염)
